# How to Add Lower Thirds in OBS

Lower thirds are one of the most commonly used graphics in live streaming.

They are typically used to display:

- Speaker names
- Locations
- Breaking news
- Event information
- Sponsor messages

## Common Methods

### 1. Static PNG Graphics

Simple and easy to use but difficult to update during a live show.

### 2. Browser Sources

Allow dynamic graphics and web-based overlays.

### 3. HTML Graphics

Provide greater flexibility and can be updated in real time.

### 4. Real-Time Graphics Systems

Often used in professional productions where graphics need to change instantly without redesigning assets.

## Conclusion

The best approach depends on your production requirements, budget, and workflow.
