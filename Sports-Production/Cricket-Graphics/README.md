# Cricket Graphics

A collection of resources, workflows, templates, and production references for creating professional cricket broadcast graphics.

This section focuses on how cricket information, live match data, and visual storytelling elements are combined to create engaging broadcast and livestream experiences.

---

## Overview

Modern cricket productions rely on real-time graphics to communicate match situations, player performance, statistics, and tournament information.

Cricket graphics workflows typically include:

* Live score presentation
* Match statistics
* Player information
* Tournament branding
* Data-driven visualizations
* Sponsor integrations
* Broadcast presentation packages

---

### Score Graphics

Resources covering cricket score presentation elements including:

* Match status graphics
* Over-by-over information
